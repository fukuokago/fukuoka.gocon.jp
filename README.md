fukuoka.gocon.jp
==

This is SPA by Nuxt.js for Go Conference'19 Summer in Fukuoka.

Build Setup
--

```sh
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build and deploy to Google Cloud Storage
$ npm run dist
```
